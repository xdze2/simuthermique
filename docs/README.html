Modèle thermique dynamique

Le modèle est l'équivalent d'un circuit électrique de résistances et condensateurs.

Système d'équation différentielles linéaires:

MdTdt=K×T(t)+S(t)M \frac{dT}{dt} = K \times T(t) + S(t) MdtdT​=K×T(t)+S(t)

avec TTT le vecteur de température des noeuds du réseau, MMM les masses thermiques, KKK la matrice de connections entre les noeuds (résitances thermiques) et SSS les termes extérieures (sources).

remarque: MMM est un vecteur ou une matrice...

Solver

- See part "3 Fully discrete approximation" in Lecture 3 - Finite Volume Discretization of the Heat Equation

  • Linear system (matrix) formulation:

∂T∂t=A T+S(t)\frac{\partial T}{\partial t} = A \, T + S(t) ∂t∂T​=AT+S(t)

Tk+1T^{k+1}Tk+1 is solved using the backslash operator

  • θ=0\theta = 0θ=0 : Forward Euler (explicit, 1st order)
  • θ=1/2\theta = 1/2θ=1/2 : Crank–Nicolson (implicit, 2nd order)
  • θ=1\theta = 1θ=1 : Backward Euler (implicit, 1st order),

Remark 2: The Crank-Nicolson scheme is second order accurate but gives slowly decaying oscillations for large eigenvalues. It is unsuitable for parabolic problems with rapidly decayingtransients. The θ=1\theta=1θ=1 scheme damps all components, and should be used in the initial steps.

solver takes: A matrix, T0 values, S time-series (sparse)
